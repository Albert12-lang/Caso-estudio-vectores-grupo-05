# Caso-estudio-vectores-grupo-05
Elaboración del los casos de estudio del grupo 5. 

## Introducción
En la elaboración de los casos de estudio se elabora el código usando el HTML y JavaScript aplicando conceptos básicos como los vectores, estructuras condicionales, ciclos for y funciones. 
El caso # 1 Tienda de Abarrotes, donde el usuario puede seleccionar productos, registrar cantidades compradas y generar una factura con el cálculo del subtotal, IVA, descuento y total a pagar.
En el caso # 2 Cajero Bancario Inteligente, el cual permite registrar depósitos y retiros para calcular el saldo disponible. 

## Objetivos
* Análisis y diseño de casos de estudios reales utilizando Vectores en JavaScript.
* Llegar a elaborar de manera visible una factura en html.

## Desarrollo
1. Caso #1 Tienda de Abarrotes.
El usuario va a seleccionar el producto que quiere llevar en la lista, visualizar el precio unitario, ingresar la cantidad que lleva de un producto y agregar el producto a la compra. Al generar la factura, el sistema recorre los vectores utilizando un ciclo (for) para calcular el subtotal de cada producto y el subtotal general de la compra. Posteriormente calcula el IVA correspondiente al 15 %, verifica mediante una condición si el subtotal supera los veinte dólares para aplicar un descuento del 5 % y finalmente obtiene el total a pagar.
2. Caso # 2 Cajero Bancario Inteligente. 
En el caso de los retiros, también verifica que exista saldo suficiente antes de realizar la operación. Cada movimiento se almacena en un vector junto con su tipo y monto, lo que permite mantener un historial de las transacciones realizadas.
Posteriormente, mediante un ciclo for, el sistema recorre el vector de movimientos para mostrar todas las operaciones registradas y calcular un resumen final con el total depositado, el total retirado y el saldo disponible en la cuenta. Para el desarrollo de este sistema se utilizaron variables, vectores, estructuras condicionales if… else, ciclos for y funciones en JavaScript.

## Conclusión
Estos casos nos permitió aplicar de manera práctica los conocimientos básicos de programación en JavaScript. Se utilizaron vectores para almacenar información, ciclos para recorrer los datos y estructuras condicionales para tomar decisiones según cada situación. Reforzó el manejo de eventos en HTML y la interacción con los elementos de la página mediante el DOM.
